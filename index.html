What is This?

an (data) API that returns random love affirmations. it comes with two properties which is "lang" and "affirmation"

Routes:

/api

returns a random affirmation

Queries:

?lang=[language/dialect]

returns a random affirmation for every language/dialect (only bisaya, english, filipino, and ilonggo are available)


/api?lang=english
